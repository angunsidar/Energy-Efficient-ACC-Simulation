Open the .m file in MATLAB and click "Run". The script will:
1. Generate a stochastic traffic scenario.
2. Simulate both Static and Dynamic vehicles.
3. Output the "Death Point" analysis to the command window.
4. Plot the comparison figures (Velocity, SoC, Efficiency).
